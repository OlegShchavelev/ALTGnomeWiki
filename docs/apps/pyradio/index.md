# pyRadio

pyRadio — интернет-радиоплеер с интерфейсом командной строки.

<AGWGallery />

## Установка из репозитория

Пакет `pyRadio` можно установить через терминал:

::: code-group

```shell[apt-get]
su -
apt-get update
apt-get install pyradio
```

```shell[epm]
epm -i pyradio
```

:::
